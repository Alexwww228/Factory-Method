## Описание

Программа имитирует процесс найма сотрудников в компании с различными отделами: **Разработчики**, **Менеджеры**, **Дизайнеры**.

Каждый отдел использует свой метод создания сотрудников (Factory Method), при этом общий процесс найма одинаков для всех.

---

## Функции

* Создание сотрудников разных типов: Developer, Manager, Designer
* Расчёт зарплаты с учётом опыта и бонусов
* Отображение информации о сотруднике и его обязанностей
* Управление списком сотрудников в отделе
* Лёгкое расширение: добавление новых типов сотрудников и отделов

---

## Архитектура

* **IEmployee** — интерфейс всех сотрудников
* **EmployeeBase** — базовый класс с общей логикой
* **Конкретные сотрудники** — реализуют специфическое поведение: Developer, Manager, Designer
* **HiringDepartment** — абстрактный отдел, который управляет процессом найма
* **Конкретные отделы** — переопределяют Factory Method для создания конкретного сотрудника

---

## Классы

| Класс                  | Назначение                                                 |
| ---------------------- | ---------------------------------------------------------- |
| `IEmployee`            | Интерфейс для всех сотрудников                             |
| `EmployeeBase`         | Общая логика сотрудников: опыт, зарплата, вывод информации |
| `Developer`            | Конкретный сотрудник-разработчик с бонусом                 |
| `Manager`              | Конкретный сотрудник-менеджер с премией                    |
| `Designer`             | Конкретный сотрудник-дизайнер                              |
| `HiringDepartment`     | Абстрактный отдел с методом найма и фабричным методом      |
| `DeveloperDepartment`  | Создаёт Developer                                          |
| `ManagementDepartment` | Создаёт Manager                                            |
| `DesignDepartment`     | Создаёт Designer                                           |
| `Program`              | Основная программа, консольный интерфейс                   |
